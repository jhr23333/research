## 信号规则

| 数据源 | 触发条件 | 动作 |
|--------|---------|------|
| 韩国半导体出口前10日同比 | 连续2月同比 < -10% | 追加至鹏鼎 假设.md H2 待核实 |

| 国内晶圆厂招标公告 | 出现 PCB/基板 相关采购 | 追加至鹏鼎 假设.md H2 待核实（潜在订单线索） |

## 说明

- 数据层：`data\` 只存原始 CSV，不做分析
- 信号层：脚本运行后自动判断是否触发阈值，触发则写入公司 `假设.md`
- 看板层：`看板\` 笔记负责可视化，通过 `![[]]` 嵌入图表
- 所有数据系列在登记表里必须填写"关联假设"，孤立数据不入库
